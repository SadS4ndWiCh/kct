Simple shell script to run progams in background.

```
KCT

usage: kct [OPTIONS] <PROGRAM>

Options:
    -h|--help   Show the usage details
    -n|--name   Set an alias name
    -d|--down   Kill the given program

Example:
    kct discord
    kct --name hell discord
    kct --down hell
    kct --help
```
